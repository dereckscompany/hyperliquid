# Generated assertion helpers

Runtime assertion helpers generated by roxyassert from typed roxygen
documentation and exported via the `@exportassert` tag.

## Usage

``` r
assert_type_Candles(value)

assert_type_Fill(value)

assert_type_FundingHistory(value)

assert_type_L2Level(value)

assert_type_MarginSummary(value)

assert_type_OrderResult(value)

assert_type_PerpMeta(value)

assert_type_Position(value)

assert_type_StakingSummary(value)

assert_type_TransferAck(value)
```

## Arguments

- value:

  The value or shaped object validated by the assert helper.

## Value

`assert_args_*` return `invisible(NULL)`; `assert_return_*` return the
validated value; `assert_type_*` return it invisibly. All are called for
the side effect of signalling an error on a contract violation.
